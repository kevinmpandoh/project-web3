# Sawah Heaven Next.js

This folder is a Next.js 16 App Router migration of the original TanStack Start app.

## Scripts

```bash
npm install
npm run dev
npm run build
npm start
```

The UI/components/game logic were copied from the original app. TanStack file routes were replaced with Next.js App Router pages, and TanStack server functions were replaced with Next API routes under `src/app/api`.
